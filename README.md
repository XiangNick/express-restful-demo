# express-restful-demo
## nodejs基于express实现原生restful接口的demo

1. clone后，进入myapp文件夹，执行`npm i`来安装依赖。
2. 在本地mysql数据库执行根目录的db.sql文件初始化数据库
3. 在myapp文件夹执行`npm run start`启动服务器，默认监听3000端
口
4. 使用http工具如postman进行测试
